# 🛒 E-Commerce Platform

A full-stack e-commerce web application built using modern technologies. This platform allows users to browse products, add items to cart, and securely place orders.

## 🌐 Live Demo
Frontend: https://e-commerce-lb64.vercel.app

---

## 🚀 Features

- 🛍️ Browse products with images and details  
- 🔍 Search and filter functionality  
- 🛒 Add to cart and manage cart items  
- 🔐 User authentication (Login / Signup)  
- 💳 Secure payment integration (Stripe)  
- 📦 Order placement system  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Tailwind CSS  
- Axios  

### Backend
- Node.js  
- Express.js  
- MongoDB  

### Payment Gateway
- Stripe  
- RazorPay
- Cash On Delivery
